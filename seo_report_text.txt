3. Technical SEO Audit
Indexation Status [CRITICAL]
• site:hoablgoa.com returns ZERO results — all 3 pages are unindexed.
• The domain appears relatively new. However, no indexation after launch is a sign of possible noindex meta tags,
robots.txt block, or GSC not set up.
• ACTION: Check page source on all 3 pages for meta name='robots' content='noindex'. Verify robots.txt
at hoablgoa.com/robots.txt.
• ACTION: Set up Google Search Console, verify domain ownership, and submit URL Inspection for all
pages.
XML Sitemap [CRITICAL]
• No sitemap.xml detected at hoablgoa.com/sitemap.xml.
• With a custom HTML site (no CMS auto-generation), sitemap must be created manually.
• ACTION: Create sitemap.xml listing all 4 URLs (homepage, gulf-of-goa, one-goa, privacy) and submit
via GSC.
Canonical Tags [HIGH]
• No canonical tags confirmed across any page.
• The Gulf of Goa page is accessible at both /gulf-of-goa and /gulf-of-goa/ (trailing slash variant) — potential
duplicate.
• ACTION: Add self-referencing canonical to all pages. Standardise trailing slash handling.
robots.txt [HIGH]
• robots.txt content could not be independently fetched. Must be manually verified.
• If Disallow: / is present, Googlebot cannot crawl anything.
• ACTION: Ensure robots.txt only blocks /images/ directory (if desired) and allows all main pages.
Video Background (hero-bg.mp4) [HIGH]
• The homepage uses an MP4 video as the hero background (video/hero-bg.mp4).
• This is a significant LCP risk on mobile — video files are never LCP candidates but delay all other resource
loading.
• ACTION: Lazy-load the video. On mobile, replace with a static WebP/AVIF image using a CSS media
query.
HTTPS / Security [GOOD]
• HTTPS confirmed on hoablgoa.com. SSL certificate active.
• No mixed-content issues detected from source analysis.
• Recommend: Add HSTS header and basic Content-Security-Policy via server config.
Mobile Responsiveness [MEDIUM]
• Navigation includes a hamburger menu (visible via JS toggle), indicating mobile-aware design.

12. Search Experience Optimisation (SXO)
Search Intent Match
Target users are HNI investors and lifestyle buyers searching for: luxury plots Goa, HOABL Goa price, sea-facing
plots Vasco. Intent is commercial investigation (comparing options, checking prices, assessing ROI). The site
correctly provides: project specs, pricing tiers, amenity lists, possession dates, and direct enquiry forms. Intent
match: GOOD.
UX & Conversion Friction Analysis
[HIGH FRICTION] All document downloads (brochure, floor plan, master plan, walkthrough) are gated behind lead
forms. While strategically correct for lead gen, the same details fields on every popup creates fatigue. Each form
requires name, phone, and project selection independently — no session persistence.
Recommendation: Implement a single sticky sidebar form that persists across scroll. Or use a 'one-tap verify' pattern
after initial submission.
[MEDIUM FRICTION] The hero section counter '500+ Happy Owners' lacks supporting evidence on the page — no
testimonials, no reviews, no case studies. For n1–4.5 Cr purchases, buyers need social proof.
Recommendation: Add a testimonials carousel with real buyer names, investment amounts, and outcome.
[MEDIUM FRICTION] Walk-through sections on project pages have a preview image but no actual embedded
video — clicking 'Play Virtual Tour' likely triggers a lead capture form rather than playing a video.
Recommendation: Embed a real YouTube walkthrough video. Gated video behind a form reduces engagement
significantly.
[POSITIVE SIGNAL] The 'GET CALLBACK IN 15 MINS' CTA on homepage is strong and time-bound — creates
urgency without being pushy. The homepage form with project selection dropdown is clean and
conversion-focused.
Recommendation: No change needed. Consider A/B testing '60 MINUTE CALLBACK' vs '15 MIN CALLBACK' — the
15 min claim must be deliverable.
[POSITIVE SIGNAL] Navigation includes direct links to both project pages from homepage. Mobile hamburger
menu present. Clear visual hierarchy.
Recommendation: No change needed. Internal linking structure is solid.
